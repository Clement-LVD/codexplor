## Example

Assuming you have a R folder full of custom R funcРђЎ :

`fn_net <- get_text_network_from_files()`

regex you provided, then construct a regex from the result (i.e.┬аpasted
together with the Рђю\|РђЮ separator and optionnally adding a suffix and
prefix defined by the user) Then, a 2nd search is made in the content to

to file path with first match) For example the default will try to catch
a basic funcРђЎ definition, extracting the text before this definition.
